## Install

```bash
npm i awspack
```

## Use

```js
import { s3, lambda } from "awspack";

const s3Client = new s3.S3Client({ region: "us-east-1" });
const lambdaClient = new lambda.LambdaClient({});
```

Future bundles may group services by domain (e.g., analytics-only, serverless-only ...)
